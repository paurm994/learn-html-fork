CSS

  • CSS significa Cascade Stylesheets (hojas de estilo en cascada). Sirve para añadir los estilos al HTML y hacer que las páginas tengan un diseño más atractivo.
  • Hay tres formas de añadir CSS a una página:
    1. inline:los estilos se colocan en la misma etiqueta de HTML usando el atributo style: 
      <p style="color:brown">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Est, beatae!</p>
    2. interno:dentro de la etiqueta <style> 
      <style>
    h1 {
     color: brown; 
    }
  </style>
    3. externo:se colocan los estilos en un archivo .css distinto que se importa con <link>.
