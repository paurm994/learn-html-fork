blockquote

HTML, según la Wikipedia es:

HTML, acrónimo en inglés de HyperText Markup Language («lenguaje de marcado de hipertexto»), hace referencia al lenguaje de marcado utilizado en la creación de páginas web.

Cita corta con q

HTML es el estándar que prevalece en la visualización de páginas web y es adoptado por todos los navegadores actuales.

abbr

HTML es el lenguaje para crear los elementos que aparecen en una web.

CSS es el lenguaje o formato habitual para dar estilo a las páginas web.

Javascript es el lenguaje de programación que se puede ejecutar en el navegador por defecto en las páginas web.
